# vaisShop Roadmap: De MVP Visual a Producción 🚀

## ✅ Fase 1: Identidad Visual y Motor 3D (COMPLETADO)
- [x] **Estructura Base:** Hydrogen + React Router 7.
- [x] **UI Premium:** Diseño "High-End" con tipografía Inter y layout optimizado.
- [x] **Realismo 3D:** Materiales de plástico pulido, sombras suaves y levitación.
- [x] **Responsive Design:** Orden de lectura optimizado para dispositivos móviles.
- [x] **Branding:** Favicon funcional y logo dinámico con `currentColor`.

## 🔗 Fase 2: Conectividad Shopify Headless (Siguiente)
- [ ] **Configuración API:** Setup de `.env` con Storefront API Access Token.
- [ ] **GraphQL Integration:** Queries para Title, Description y Price dinámicos.
- [ ] **Sincronización de Variantes:** Vincular los colores de Shopify a los materiales del modelo 3D.
- [ ] **Cart Context:** Implementar el provider de carrito de Hydrogen.

## 🎨 Fase 3: Personalización e Interactividad
- [ ] **Color Picker:** Interfaz para cambio de color en tiempo real.
- [ ] **Micro-interacciones:** Animaciones con Framer Motion para el texto y transiciones.
- [ ] **Vistas 360:** Hotspots interactivos sobre el modelo para mostrar specs técnicos.

## 🛠️ Fase 4: Optimización y Lanzamiento
- [ ] **Performance:** Compresión Draco para el modelo 3D.
- [ ] **Deployment:** Despliegue en Shopify Oxygen.
